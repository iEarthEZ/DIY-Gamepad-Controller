# DIY-Gamepad-Controller
Trying to build a gamepad controller from scratch using kicad. I will (as heard of) need the pcb to be shipped to me so i can code some arduino probably but that will be on my cost and i'll finish everything and keep it open sourced (although its alot to cover for beginners).

This project is about a custom gamepad controller made in kicad. This github repositor includes all the schematic and pcb. The pcb is a square that can be used in any custom 3d printed cad of your choice. CAD or Computer-Aided Design is basically what your case of the controller will be. I've provided the basic pcb so you can download the gers.zip folder and get the pcb shipped for yourself from jlcpcb, pcbway or any other of your choice, you can even tweak the main pcb board for yourself if you like. The thing is i shipped you the project files and pcb so you can enjoy the controller you make as you like! In the project its all infront of you so even if you are a beginner and know nothing about pcb you'll understand what does what. As of now im currently busy irl but i will make a polished cad(as a default one if you are struggling finding a cad or making one) and i will code the arduino micro pro.
