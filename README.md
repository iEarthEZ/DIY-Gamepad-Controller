# DIY-Gamepad-Controller
Making a custom gamepad controller from scratch. {I'll update code files and firmware when my parts get shipped.}.

Lets walk you through here from where i got the idea to where we are now. First things first the idea came to my mind because not too long ago i bought my cool red ps4 controller so i thought "what if i could make something like this myself?". Thats what it hit me, i was like I can make that so i got to work on the research my brother was making a split keyboard and he used the arduino pro micro mini as the "brain" or main chip. He said if you are making something like that use the pro micro mini so i atlast settled down with the normal usb-c version of the pro micro(the mini's pins are abit different). If you are making this project like me i recommend the video https://www.youtube.com/watch?v=mvv1BRjswew by Voltaat. It will get you started with ide and programming so it'll be easy and i recommend watching some soldering tutorials lol. Also first things first you should download all the project files and open them up in kicad and check it out or edit stuff yourself!(feel free). If you goto the pcb section and press alt + 3 you can see a base model of how it will look(its the reference image at the start of this readme). Its straight forward, upload gerber files to jlcpcb or pcbway or anyother you like i guess, wait for the pcb to get shipped to you, order the given parts in the BOM and just assemble them. Thats it! I made the controller a square but you can make it any shape you like. If you know the basics of kicad i don't think it will be a pain to get through everything. It was my second ever pcb and project with kicad and stuff so i was pretty scared but at the end i found out it ain't hard. Looking forward to everyone that makes this project and in the next upd i'll drop all the firmwarer and code files even a tutorial on how to code the ide after my parts gets shipped. Till then. Peace.
